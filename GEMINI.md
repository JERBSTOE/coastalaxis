This is a website for Coastal Axis Insurance Agency, owned by Jared. The site is a static HTML/CSS/JS website hosted on GitHub Pages.

## File Structure
- `index.html` - Homepage (hero section, value props, insurance overview)
- `why-coastal-axis.html` - Why choose Coastal Axis (independent vs captive messaging)
- `about.html` - About Jared (bio, experience, background)
- `personal-insurance.html` - Personal insurance types (Auto, Home, Landlord, Earthquake, Flood)
- `business-insurance.html` - Business insurance types (Workers' Comp, General Liability, Commercial Auto, Professional Liability)
- `contact.html` - Contact form (powered by Formspree, sends to jared@coastalaxis.com)
- `css/style.css` - All styling (colors: navy #1B3A5C, gold #C8962E)
- `js/main.js` - Mobile menu and navigation behavior
- `images/coastal-axis.png` - Company logo

## Guidelines
- Use simple, professional language appropriate for an insurance agency
- Keep the existing design style and color scheme (navy and gold)
- The navigation bar, header, and footer are the same across all pages - if updating these, update all HTML files
- When asked for text/copy changes, provide them as a Pull Request
- Do not change the Formspree form action URL in contact.html
- The site is deployed via GitHub Pages from the main branch
